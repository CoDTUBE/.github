# CoDTUBE

Welcome to the official GitHub organization of **CoDTUBE**.

CoDTUBE is a community-driven platform built for Call of Duty players to discover, share and preserve great gameplay moments. Alongside the platform itself, this organization hosts open source projects, tools and integrations developed for the community.

## Projects

- Android App (Coming soon)
- OBS Studio Integration (https://github.com/CoDTUBE/OBS-Uploader)

## Website

https://codtube.tv

## Open Source

Not every part of CoDTUBE is open source, but whenever it makes sense, we enjoy sharing tools with the community.

If you have ideas, find a bug or want to contribute to one of our public projects, feel free to open an issue or submit a pull request.

## Contributing

Whether you are improving, fixing bugs or building new features, every contribution is appreciated.
